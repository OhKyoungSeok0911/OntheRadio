# 카카오페이 QR 송금 앱

QR 코드를 스캔하여 카카오페이로 송금하는 React 앱입니다.

## 기능

- QR 코드 스캔으로 계좌 정보 읽기
- 수동으로 계좌 정보 입력
- 카카오페이 송금 링크 생성
- QR 코드 생성 및 표시
- 카카오페이 앱 연동 (모바일/데스크톱 지원)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 환경 변수 설정
cp .env.example .env
# .env 파일을 열어서 카카오페이 API 키를 입력하세요

# 개발 서버 실행
npm start
```

브라우저에서 http://localhost:3000 으로 접속하세요.

## 카카오페이 API 키 발급 방법

### 1. 카카오디벨로퍼스 가입 및 애플리케이션 등록

1. [카카오디벨로퍼스](https://developers.kakao.com/)에 접속
2. 카카오 계정으로 로그인 (없으면 회원가입)
3. 상단 메뉴에서 **"내 애플리케이션"** 클릭
4. **"애플리케이션 추가하기"** 버튼 클릭
5. 다음 정보 입력:
   - **앱 이름**: 원하는 앱 이름 (예: "QR 결제 앱")
   - **사업자명**: 개인/회사명
   - **카테고리**: 적절한 카테고리 선택
6. **"저장"** 클릭

### 2. REST API 키 확인

1. 생성한 애플리케이션 선택
2. 좌측 메뉴에서 **"앱 키"** 클릭
3. **"REST API 키"** 복사 (이것이 `REACT_APP_KAKAO_REST_API_KEY`에 들어갈 값)

### 3. 카카오페이 결제 API 활성화

1. 좌측 메뉴에서 **"제품 설정"** > **"카카오페이"** 클릭
2. **"카카오페이 활성화"** 버튼 클릭
3. 가맹점 코드(CID) 발급:
   - **테스트용**: `TC0ONETIME` (자동 발급)
   - **실제 운영용**: 사업자 등록 후 발급 필요

### 4. 플랫폼 설정

1. 좌측 메뉴에서 **"앱 설정"** > **"플랫폼"** 클릭
2. **"Web 플랫폼 등록"** 클릭
3. 사이트 도메인 입력:
   - 개발: `http://localhost:3000`
   - 운영: 실제 도메인 (예: `https://yourdomain.com`)

### 5. 환경 변수 설정

프로젝트 루트에 `.env` 파일을 생성하고 다음 내용을 입력:

```bash
# .env 파일
REACT_APP_KAKAO_REST_API_KEY=발급받은_REST_API_키
REACT_APP_KAKAO_APP_KEY=발급받은_앱_키
```

**예시:**
```bash
REACT_APP_KAKAO_REST_API_KEY=abc123def456ghi789jkl012mno345pq
REACT_APP_KAKAO_APP_KEY=xyz789abc123def456ghi789jkl012mno
```

### 6. 개발 서버 재시작

환경 변수를 변경한 후에는 개발 서버를 재시작해야 합니다:

```bash
# 서버 중지 (Ctrl+C)
# 서버 재시작
npm start
```

## 주의사항

- **`.env` 파일은 Git에 커밋하지 마세요** (이미 .gitignore에 포함되어 있습니다)
- **REST API 키는 절대 공개하지 마세요** (보안 위험)
- **테스트 환경**에서는 `TC0ONETIME` 가맹점 코드 사용 가능
- **실제 결제**를 위해서는 사업자 등록 및 카카오페이 가맹점 승인이 필요합니다

## 사용 방법

1. **QR 코드 스캔**: "QR 코드 스캔" 버튼을 눌러 카메라로 QR 코드를 스캔합니다.
2. **수동 입력**: 계좌 정보를 직접 입력할 수 있습니다.
3. **송금 링크 생성**: 계좌 정보를 입력한 후 "카카오페이 링크 생성" 버튼을 클릭합니다.
4. **QR 코드 표시**: 생성된 QR 코드를 다른 사람에게 보여줄 수 있습니다.

## QR 코드 형식

QR 코드는 다음 형식 중 하나를 지원합니다:

- JSON 형식: `{"bank": "은행명", "accountNumber": "계좌번호", "accountHolder": "예금주", "amount": "금액"}`
- 파이프 구분 형식: `은행명|계좌번호|예금주|금액`

## 주의사항

- 실제 카카오페이 송금을 위해서는 카카오페이 API 키가 필요합니다.
- 현재는 데모 버전으로 링크 생성만 지원합니다.
- 카메라 접근 권한이 필요합니다.
